GDWR's Nix Flake
================

Quickstart
----------
See flake outputs: `nix flake show github:gdwr/nixcfg` \
Install host: `nixos-rebuild switch --flake github:gdwr/nixcfg#desktop` \
Apply home-manager config: `home-manager switch --flake github:gdwr/nixcfg#gdwr`
